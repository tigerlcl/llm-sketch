### Rule Mining
The rule to determine the 'Total_Stops' based on 'Route':
```python
def determine_stops(route):
    segments = route.split(' → ')
    num_stops = len(segments) - 1
    if num_stops == 0:
        return 'non-stop'
    elif num_stops == 1:
        return '1 stop'
    else:
        return f'{num_stops} stops'
```
This function splits the 'Route' by the arrow symbol, counts the segments, and then determines the number of stops accordingly.
