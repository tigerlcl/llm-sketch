### Step 3: Rule Mining

To derive the rule for calculating `Duration`, we can use the `Dep_Time` and `Arrival_Time`. Here's a Python function that calculates the duration based on these times:

**Python Code Block for Rule Mining:**
```python
from datetime import datetime, timedelta

def calculate_duration(dep_time, arrival_time):
    # Convert times to datetime objects
    dep_datetime = datetime.strptime(dep_time, '%H:%M')
    # Handle arrival times that roll over to the next day
    if ' ' in arrival_time:
        arrival_time = arrival_time.split(' ')[0]
    arrival_datetime = datetime.strptime(arrival_time, '%H:%M')
    
    # Adjust for flights that arrive after midnight
    if arrival_datetime < dep_datetime:
        arrival_datetime += timedelta(days=1)
    
    # Calculate duration
    duration = arrival_datetime - dep_datetime
    hours = duration.seconds // 3600
    minutes = (duration.seconds // 60) % 60
    
    return f'{hours}h {minutes}m'

# Example usage:
print(calculate_duration('19:50', '22:50'))  # Output should be '3h 0m'
```

This function handles the basic case where the flight does not span multiple days significantly. For more complex cases, additional logic to parse the date might be necessary.
